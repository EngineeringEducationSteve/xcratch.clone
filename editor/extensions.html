There are several special extensions that we can use in this tool. Copy the URL of the extension you need and paste it back into our Scratch Port. This is a library so you'll now have extra blocks you can use to control this toy.

  • Princess Peach: https://sjcoefablab.org/extensions/legopeach.mjs
  • Mario: https://sjcoefablab.org/extensions/legomario.mjs
  • Luigi: https://sjcoefablab.org/extensions/legoluigi.mjs
  • Bluetooth Low Energy: https://sjcoefablab.org/extensions/legoble.mjs
  • LEGO Spike Essentials: https://sjcoefablab.org/extensions/spikeessential.mjs
Do not try to go to the link! Just cut and paste it back into the Scratch Editor.
